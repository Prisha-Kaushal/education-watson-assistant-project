# Education Website with Watson Assistant

This project is an education-based website developed using HTML, CSS, and JavaScript. 
It integrates IBM Watson Assistant to provide an AI chatbot that answers student queries. 
The chatbot is also connected to WhatsApp using Twilio to allow users to interact and receive instant support.

## Technologies Used
- HTML
- CSS
- JavaScript
- IBM Watson Assistant
- Twilio WhatsApp API
- 
## Features
- Education information website
- AI chatbot support
- WhatsApp chatbot integration
